# Warehouse - 論文リスト
 
読んだ論文・気になる論文をタイトル・著者・発表年とともに時系列順で整理するリポジトリです。
 
---
 
## 使い方
 
以下のフォーマットに従って論文を追加してください。新しい年度のセクションが必要な場合は、見出しを追加してください。
 
```
| タイトル | 著者 | メモ |
|---------|------|------|
| Example Paper Title | Author et al. | 簡単なメモ（任意） |
```
